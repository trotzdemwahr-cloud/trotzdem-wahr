2. Hosting

Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf der Website verarbeitet GitHub technische Informationen, die für die Bereitstellung der Website erforderlich sind. Hierzu können insbesondere die IP-Adresse, Datum und Uhrzeit des Zugriffs, der verwendete Browser, das Betriebssystem sowie weitere technische Informationen gehören. Die Verarbeitung erfolgt, um die Website sicher und zuverlässig bereitzustellen.

3. Server-Logfiles

Beim Besuch dieser Website werden durch den Hostinganbieter automatisch Informationen in sogenannten Server-Logfiles gespeichert. Hierzu können unter anderem gehören:

  • IP-Adresse
  • Datum und Uhrzeit des Zugriffs
  • aufgerufene Seite
  • Browsertyp und Browserversion
  • Betriebssystem
  • Referrer-URL (sofern übermittelt)

Diese Daten dienen ausschließlich der technischen Bereitstellung, der Sicherheit und der Fehleranalyse der Website. Eine Zusammenführung dieser Daten mit anderen Datenquellen erfolgt nicht.

4. Google Fonts

Auf dieser Website werden Schriftarten von Google Fonts verwendet. Beim Aufruf der Website kann eine Verbindung zu Servern von Google hergestellt werden, um die Schriftarten bereitzustellen. Dabei können technische Informationen, wie beispielsweise deine IP-Adresse, an Google übermittelt werden. Weitere Informationen zum Datenschutz bei Google findest du unter: https://policies.google.com/privacy
